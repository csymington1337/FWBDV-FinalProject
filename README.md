## âœ¨ Features

### Responsive Design
- **Mobile-First Approach** - Optimized for smartphones, tablets, and desktops
- **Breakpoints:**
  - Mobile: Default (< 480px)
  - Tablet: 480px+
  - Desktop: 1024px+
- **Hamburger Navigation** - Collapsible mobile menu with smooth transitions

### Interactive Class Schedule
- Day-based filtering with toggle buttons
- Responsive layouts:
  - Mobile: Card-based view with stacked information
  - Tablet/Desktop: Traditional table format with fixed columns
- Dynamic content display for different screen sizes

### Contact Form
- **Form Validation:**
  - Required field checking
  - Email format validation (regex: `/^[^\s@]+@[^\s@]+\.[^\s@]+$/`)
  - Member status toggle
  - Inquiry reason and fitness goal selectors
- **Visual Feedback:**
  - Red asterisks for required fields
  - Alert messages for validation errors
  - Success confirmation on submission

### Membership Features
- Interactive plan comparison
- Toggle between membership tiers
- Styled radio buttons with custom appearance
- Clear pricing and benefits display

### Location & Contact
- Embedded Google Maps
- Business hours display
- Contact information (phone, email, address)
- Icon-based visual presentation
